PromiseInterface runQuery(QueryBuilderInterface $query)

Runs the given querybuilder on the underlying driver instance. However the query will be ignored, only the parameters are used.

The driver CAN throw an exception if the given querybuilder is not supported. An example would be a SQL querybuilder and a Cassandra driver.
